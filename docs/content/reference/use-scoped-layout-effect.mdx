## Overview

The `useScopedLayoutEffect` hook runs side effects synchronously after DOM updates, similar to `React.useLayoutEffect` but with reactive scope support.

## Function Signature

```typescript
function useScopedLayoutEffect(
  effect: (scope: Scope) => Destructor,
  dependencies?: DependencyList
): void
```

## Parameters

- `effect`: A function that receives a `Scope` and can return a cleanup function
- `dependencies`: Optional dependencies that control when the effect re-runs

## Description

`useScopedLayoutEffect` is similar to `useScopedEffect` but runs synchronously after DOM mutations, before the browser paints. Use this for DOM measurements or synchronous DOM updates.

## Examples

```typescript
useScopedLayoutEffect(
  (scope) => {
    const value = impulse.getValue(scope)
    // Perform synchronous DOM operations
    element.style.width = `${value}px`

    return () => {
      // Cleanup synchronous DOM changes
    }
  },
  []
)
```
